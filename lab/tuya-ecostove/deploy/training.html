🚀 Part 1 — เริ่มต้นใช้งาน

ลงทะเบียน เข้าสู่ระบบ ทำความรู้จักหน้าจอ

📝 ลงทะเบียนแอดมิน
  1. เปิดเว็บ biomassstove.vercel.app
  2. กด "ลงทะเบียนแอดมินใหม่"
  3. กรอก ชื่อ-นามสกุล + อีเมล
  4. ตั้ง PIN ส่วนตัว (ตัวเลข 4-8 หลัก)
  5. ใส่ Invite Code ที่ได้รับ
  6. ได้ Recovery Code — จดเก็บไว้ให้ดี!
⚠️ Recovery Code ใช้ได้ครั้งเดียว!
เมื่อใช้ reset PIN แล้ว code จะถูกลบทันที → ต้อง login แล้วไปสร้างใหม่ที่ "สร้าง Recovery Code"
🔐 เข้าสู่ระบบ + จัดการ PIN
เข้าสู่ระบบ
  1. กรอก อีเมล ที่ลงทะเบียน
  2. กรอก PIN ส่วนตัว
  3. เข้าหน้าแอดมินได้เลย
เปลี่ยน PIN
  • ไปที่แท็บ "เปลี่ยน PIN" → ใส่ PIN เก่า → ตั้ง PIN ใหม่
ลืม PIN?
  1. กด "ลืม PIN" ที่หน้า login
  2. ใส่ อีเมล + Recovery Code
  3. ตั้ง PIN ใหม่
⚡ หลัง reset ต้องสร้าง Recovery Code ใหม่!
Login → เมนู "สร้าง Recovery Code" → ถ้าไม่สร้าง ลืม PIN อีกที ต้องให้ Super Admin reset

  • 📊 ภาพรวม — สถานะเซนเซอร์ เซสชันวันนี้ API quota
  • ✅ รออนุมัติ — ตรวจสอบ + อนุมัติ/ปฏิเสธ session
  • 📋 ข้อมูล — ดูข้อมูลเซนเซอร์ + TVOC/CO + Export Excel
  • 📈 ข้อมูลเบื้องต้น — กราฟค่าเฉลี่ย เปรียบเทียบเตาเก่า/ใหม่
  • 🔬 เชิงลึก — วิเคราะห์สถิติ แผนที่ AQI กราฟละเอียด
  • 🏠 กลุ่มตัวอย่าง — จัดการบ้าน GPS จังหวัด/อำเภอ/ตำบล
  • 📡 อุปกรณ์ — จัดการเซนเซอร์ ผูกบ้าน ค้นหาเครื่องใหม่
  • 👥 อาสา — ลงทะเบียนอาสาสมัคร
  • 📁 โครงการ — สร้าง/แก้ไขโครงการวิจัย
  • 📅 ช่วงเก็บข้อมูล — ตั้งวันเริ่ม-สิ้นสุด ประเภทเตา
  • 📖 คู่มือ — คู่มือใช้งานในตัว
📋 ลำดับการตั้งค่า

ต้องทำตามลำดับนี้ เพราะข้อมูลอ้างอิงกัน:

📁 1. โครงการ → 👥 2. อาสา → 🏠 3. บ้าน → 📡 4. ผูกเซนเซอร์ → 📅 5. ช่วงเก็บ
💡 ตั้งค่าครั้งเดียว — ใช้งานประจำวันจะใช้แค่: ภาพรวม → รออนุมัติ → ดูข้อมูล

⚙️ Part 2 — ตั้งค่าระบบ

โครงการ อาสา บ้าน อุปกรณ์ ช่วงเก็บข้อมูล

📁 สร้างโครงการ
  1. ไปที่แท็บ "โครงการ"
  2. กด "เพิ่มโครงการ"
  3. กรอก ชื่อโครงการ + คำอธิบาย
  4. กด บันทึก
👥 เพิ่มอาสาสมัคร
  1. ไปที่แท็บ "อาสา"
  2. กด "เพิ่มอาสาสมัคร"
  3. กรอก ชื่อ-นามสกุล + เลือก โครงการ
  4. กรอก เบอร์โทร (ถ้ามี)
  5. กด บันทึก
⚡ ต้องสร้างอาสาก่อนสร้างบ้าน เพราะตอนเพิ่มบ้าน ต้องเลือกอาสาที่รับผิดชอบ
🏠 เพิ่มบ้านกลุ่มตัวอย่าง
  1. ไปที่แท็บ "กลุ่มตัวอย่าง"
  2. กด "เพิ่มบ้าน"
  3. กรอก ชื่อบ้าน + เลือก โครงการ + อาสา
  4. กรอก ที่อยู่: จังหวัด / อำเภอ / ตำบล
  5. ตั้ง พิกัด GPS (latitude, longitude)
  6. กด บันทึก
📡 เซนเซอร์ตรวจพบอัตโนมัติ — ไม่ต้องติ๊ก "มีเซนเซอร์" → ระบบจะตรวจเองเมื่อผูกอุปกรณ์
📡 จัดการอุปกรณ์
  1. ไปที่แท็บ "อุปกรณ์"
  2. ดูรายชื่อเครื่องทั้งหมดจาก Tuya Cloud
  3. กด "ผูกบ้าน" เพื่อเลือกบ้านให้เครื่อง
  4. ระบบจะอัพเดท "มีเซนเซอร์" ให้อัตโนมัติ
ชื่อ 2 แบบ
  • BS 001 — ชื่ออุปกรณ์ (ตั้งเอง, ไม่ผูกบ้าน ย้ายได้)
  • MT15 001 — ชื่อเซนเซอร์ (จากรุ่นใน Tuya)
📱 ลงทะเบียนเซนเซอร์ใหม่

กรณีได้เซนเซอร์ตัวใหม่:

ขั้นตอนที่ 1: Pair ในแอป Smart Life
  1. เปิดแอป Smart Life บนมือถือ
  2. Login ด้วยบัญชีโครงการ
  3. ตั้ง Hotspot: ชื่อ Biomass / รหัส Biomass2026 / 2.4 GHz
  4. กดเพิ่มอุปกรณ์ → Add Device → ทำตามขั้นตอนในแอป
ขั้นตอนที่ 2: ลงทะเบียนในระบบ
  1. ไปที่แท็บ "อุปกรณ์"
  2. กด "ค้นหาเซนเซอร์ใหม่จาก Tuya"
  3. เลือกเซนเซอร์ที่พบ → ตั้งชื่อ
  4. กด ลงทะเบียน
📅 ตั้งช่วงเก็บข้อมูล
  1. ไปที่แท็บ "ช่วงเก็บข้อมูล"
  2. กด "เพิ่มช่วง"
  3. เลือก บ้าน ที่ต้องการ
  4. เลือก วันเริ่ม + วันสิ้นสุด
  5. เลือกประเภทเตา: eco (เตาใหม่) หรือ old (เตาเก่า)
  6. กด บันทึก
⚠️ ห้ามซ้อนวัน! บ้านเดียวกัน ห้ามมีช่วงเก็บข้อมูล 2 ช่วงที่วันทับกัน
💡 ทำไมแยก eco / old? ระบบเก็บแยกเพื่อเปรียบเทียบ — เตาใหม่ลดมลพิษได้เท่าไหร่เทียบกับเตาเก่า

📆 Part 3 — การใช้งานประจำวัน

ภาพรวม อนุมัติข้อมูล ดูรายงาน

📊 ภาพรวม (Overview)

หน้าแรกหลัง login — สรุปสถานะทั้งระบบ:

  • 📡 สถานะเซนเซอร์ — ออนไลน์ / ออฟไลน์ / กำลังเก็บ
  • 📈 เซสชันวันนี้ — กี่บ้านเก็บแล้ว / เหลือกี่บ้าน
  • 🔑 API Quota — เหลือ API call เท่าไหร่ในเดือนนี้
💡 ดูทุกวัน — เช็คเซนเซอร์ทำงานปกติ + quota พอใช้
✅ อนุมัติข้อมูล
  1. ไปที่แท็บ "รออนุมัติ"
  2. ดูการ์ด session — แบ่งตาม วันที่
  3. ตรวจสอบ: จำนวน readings, ระยะเวลา, ค่าผิดปกติ
  4. กด "อนุมัติ" หรือ "ปฏิเสธ"
สถานะ session
  • 🔵 กำลังเก็บ — session ยังไม่จบ
  • 🟢 เสร็จสมบูรณ์ — ครบ 130 นาที
  • 🟡 ไม่สมบูรณ์ — หยุดก่อนครบ
  • ✅ อนุมัติแล้ว — แอดมิน approve
  • ❌ ปฏิเสธ — ข้อมูลไม่ผ่าน
📋 ดูข้อมูล + Export
  1. ไปที่แท็บ "ข้อมูล"
  2. เลือก บ้าน + ช่วงวันที่
  3. ดูข้อมูลเซนเซอร์: PM2.5, CO2, อุณหภูมิ, ความชื้น ฯลฯ
  4. ดูข้อมูล TVOC / CO ที่กรอกมือ
  5. กด "Export Excel" เพื่อดาวน์โหลด
ข้อมูลที่เก็บได้
  • 🌡️ อุณหภูมิ (°C) / 💧 ความชื้น (%)
  • 💨 PM2.5 (µg/m³) / 🌫️ PM10 (µg/m³)
  • 🏭 CO2 (ppm) / ⚗️ HCHO (µg/m³)
  • ☁️ TVOC (กรอกมือ) / 💀 CO (กรอกมือ)
📈 Dashboard
  • 📈 ข้อมูลเบื้องต้น (Basic) — ค่าเฉลี่ยรายวัน, เปรียบเทียบเตาเก่า vs เตาใหม่, impact ของเตาใหม่
  • 🔬 เชิงลึก (Deep Insights) — วิเคราะห์สถิติ, เปรียบเทียบรายบ้าน, แผนที่ AQI, time-series
💡 เตาใหม่ (eco) คือพระเอก! Dashboard แสดงว่าเตาใหม่ดีกว่าเตาเก่าอย่างไร — เตาเก่า (old) เป็นแค่ baseline เทียบ

  2. อนุญาตตำแหน่ง (GPS) — ระบบจะหาบ้านที่ใกล้ที่สุด

  4. เลือก "กรอก TVOC/CO" หรือ "ข้ามไป"
  5. กด "เริ่มเก็บข้อมูล"

🔧 Part 5 — Troubleshoot

แก้ปัญหาที่พบบ่อย

📶 WiFi / Hotspot ไม่เชื่อมต่อ
  • ชื่อ Hotspot ถูกต้อง: Biomass (ตัว B ใหญ่)
  • รหัสถูกต้อง: Biomass2026
  • ใช้ 2.4 GHz (ไม่ใช่ 5 GHz)
  • เซนเซอร์อยู่ ใกล้มือถือ (ไม่เกิน 5 เมตร)
  • ลอง ปิด-เปิด Hotspot ใหม่
  • ลอง ปิด-เปิดเซนเซอร์ ใหม่ (กดค้าง 3 วิ)
⚠️ สาเหตุที่พบบ่อยสุด: มือถือตั้งเป็น 5 GHz โดยอัตโนมัติ!
วิธีแก้: ตั้งค่า Hotspot → เลือก Band → 2.4 GHz
📡 ค่าเซนเซอร์ค้าง / ผิดปกติ

ค่า CO2/CO ค้างสูง ไม่ลดลง แม้อากาศปกติ:

  1. กดปุ่มบนเซนเซอร์ 3 ครั้ง (Calibrate)
  2. รอ 60 วินาที ให้ค่า reset
  3. ค่ากลับมาปกติ
⚡ ป้องกัน: อย่าจ่อเซนเซอร์ใกล้แหล่งควันเข้มข้น — วางห่างจากเตาอย่างน้อย 1-2 เมตร

สัญญาณว่าค่าค้าง:

  • CO2 > 2,000 ppm ตอนอากาศปกติ
  • CO > 5 ppm ตอนอากาศปกติ
  • ค่าไม่ลดลงเลย > 30 นาที
❓ เซสชันไม่ขึ้น / ข้อมูลไม่เข้า
  • เช็ค Hotspot เปิดอยู่ ตลอดเวลาเก็บข้อมูล
  • เช็ค เซนเซอร์ online ที่หน้า "ภาพรวม"
  • เช็ค ช่วงเก็บข้อมูล ตรงกับวันนี้หรือเปล่า
  • เช็ค ครบ 2 เซสชัน/วัน แล้วหรือยัง
  • เช็ค API quota — ถ้าหมด จะไม่ sync
⚡ ยังไม่ขึ้น? รอ 5-10 นาที (ระบบ sync ทุก 5 นาที) → ถ้ารอ 15 นาทีแล้วยังไม่มีข้อมูล → แจ้งแอดมิน
