📋 วิธีวิเคราะห์ข้อมูล (Methodology)

▼

1. การเก็บข้อมูล

  • เซนเซอร์: MT13W (PM2.5, PM10, CO2, HCHO, Temp, Humidity, AQI)
  • ความถี่: ทุก 5 นาที ตลอดช่วงหุงต้ม (~2 ชม./session)
  • 10 ครัวเรือน (5 เตาดั้งเดิม + 5 เตาชีวมวล) · 3 เดือน/กลุ่ม

2. การหักค่าพื้นฐาน (Baseline Subtraction)

  • ค่าเฉลี่ย 10 นาทีแรก ของแต่ละ session = ambient baseline
  • ค่าสุทธิ (Net) = ค่าที่วัดได้ − ambient baseline ของ session นั้น
  • ค่าติดลบถูกปรับเป็น 0 · Readings ช่วง baseline ไม่นำมาวิเคราะห์

3. ข้อจำกัด

  • เก็บข้อมูลคนละช่วงเวลา (ไม่ใช่ paired design)
  • Semiconductor sensor มีความคลาดเคลื่อนมากกว่า gravimetric method
