กรุณากรอกข้อมูลเพื่อตั้งค่าระบบให้พร้อมใช้งาน

ทำไมต้องกรอกแบบฟอร์มนี้?

ระบบ BiomassStove กำลังย้ายจากบัญชีส่วนตัวของผู้พัฒนาไปยังบัญชีกลางของโครงการ เพื่อให้ทีมโครงการจัดการได้เอง ข้อมูลด้านล่างจะใช้ในการตั้งค่าบัญชีและสิทธิ์การเข้าถึงทั้งหมด

2 เบอร์โทรศัพท์สำหรับ Recovery

ปัจจุบันยังใช้ของผู้พัฒนาอยู่ — จะเปลี่ยนเป็นของผู้ดูแลโครงการเมื่อส่งมอบ

เอาไปใช้ทำอะไร?
  • กู้คืนบัญชี Gmail — กรณีลืมรหัสผ่านหรือบัญชีถูกล็อก
  • ยืนยันตัวตน Google — เมื่อเข้าสู่ระบบจากอุปกรณ์ใหม่
  • Two-Factor Authentication (2FA) — รับรหัส OTP เพื่อความปลอดภัย
เบอร์โทรศัพท์ของผู้ดูแลโครงการ
อีเมลสำรอง (Recovery Email)
ใช้กู้คืนบัญชีโครงการกรณีฉุกเฉิน

4 ระบบแอดมิน (Dashboard)

ตั้งค่าผู้ดูแลระบบสำหรับหน้า Dashboard จัดการข้อมูล

ระบบแอดมินมี 2 ระดับ
Super Admin จัดการทุกอย่าง — ลบข้อมูล, เพิ่ม/ลบ/รีเซ็ตรหัสแอดมินอื่น, เปลี่ยนสิทธิ์
Admin ดูข้อมูล, จัดการเซนเซอร์ — แต่ลบข้อมูลและจัดการแอดมินคนอื่นไม่ได้
วิธีเริ่มใช้งาน: สมัครเองได้ที่หน้า Dashboard ด้วย Invite Code ด้านล่าง — ผู้พัฒนาจะเลื่อนขั้นเป็น Super Admin ให้ภายหลัง
รหัสเข้าร่วม (Invite Code)

ใช้สำหรับให้คนอื่นลงทะเบียนเป็น Admin ด้วยตนเอง — แจกรหัสนี้ให้คนที่ต้องการเข้าถึง Dashboard
ใครมีรหัสนี้จะสมัครเป็น Admin ได้เลย
Super Admin เปลี่ยนได้เองที่ Dashboard → ปุ่ม "Invite Code"
