รหัสในระบบ

รหัสที่ใช้ในระบบ (3 ประเภท)

Personal PIN

  • ใช้ login คู่กับ Email
  • ตั้งเองตอนสมัคร
  • อย่างน้อย 4 ตัว (ตัวเลข ตัวอักษร หรือผสมก็ได้)
  • เปลี่ยนเองได้ใน Dashboard
  • เป็นส่วนตัว ไม่แชร์กับใคร

Invite Code

  • ใช้ตอนลงทะเบียนเป็น Admin ใหม่
  • รหัสกลาง แชร์ให้คนที่ต้องการเข้าระบบ
  • ใช้ได้หลายครั้ง หลายคน
  • Super Admin เปลี่ยนได้ที่ Dashboard → ปุ่ม "Invite Code"

Recovery Code

  • ใช้กู้คืน PIN ที่ลืม
  • ได้ตอนสมัคร (แสดงครั้งเดียว)
  • เป็นส่วนตัว ต้องจดเก็บไว้
  • สร้างใหม่ได้ (ต้อง login อยู่)

คำถามสำหรับทีมโครงการ

  1. อีเมลโครงการ — ปัจจุบันใช้อันนี้อยู่ จะเปลี่ยนมั้ย? biomassstove.cmru@gmail.com ใช้เข้าสู่ระบบทุก platform (Tuya, Supabase, Vercel, LINE)
  2. เบอร์โทร Recovery — เบอร์ของผู้ดูแลโครงการ ใช้กู้คืน Gmail ของโครงการ
  3. อีเมลสำรอง (Recovery Email) — สำหรับกู้คืน Gmail โครงการ ต้องเป็นคนละอันกับอีเมลโครงการ
  4. รหัสผ่านบัญชีโครงการ — ใช้ค่าเริ่มต้นได้ หรือต้องการเปลี่ยน? BioMass@cmru2026 รหัสผ่านนี้ใช้เข้าถึงทุก platform ของโครงการ
  5. รหัสเข้าร่วม (Invite Code) — ใช้ค่าเริ่มต้นได้ หรือต้องการเปลี่ยน? biomassstove2026 รหัสสำหรับให้คนอื่นลงทะเบียนเป็น Admin ด้วยตนเอง
  6. ชื่อเซนเซอร์ — BS-01 ถึง BS-10 หรือต้องการรูปแบบอื่น? แสดงบนสติกเกอร์แปะเซนเซอร์ และในระบบ Dashboard
  7. โดเมน (ที่อยู่เว็บไซต์) — ใช้ .vercel.app (ฟรี) หรือ .org (~350-500 บาท/ปี)? ใช้เข้าถึง Dashboard + ทำ QR Code แปะสติกเกอร์ที่เซนเซอร์
