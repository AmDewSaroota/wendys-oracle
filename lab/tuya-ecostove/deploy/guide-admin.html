ระบบจัดการข้อมูล Biomass Stove

📑 สารบัญ

1. สมัครแอดมิน 2. เข้าสู่ระบบ 3. ตั้งค่าเริ่มต้น (โครงการ → อาสา → บ้าน → เซนเซอร์) 4. ตั้งช่วงเก็บข้อมูล (เตาเก่า / เตาใหม่) 5. อนุมัติข้อมูล 6. เปลี่ยน PIN / Recovery Code 7. จัดการแอดมิน (Super Admin)
1

สมัครแอดมิน

ลงทะเบียนด้วยรหัสเชิญ

📝 ขั้นตอน

  1. เปิดเว็บ biomassstove.vercel.app
  2. กด "สมัครสมาชิก"
  3. กรอก อีเมล ที่ใช้ประจำ
  4. ตั้ง PIN ส่วนตัว (อย่างน้อย 4 ตัว)
  5. ใส่ รหัสเชิญ ที่ได้รับจากผู้ดูแล
  6. กด สมัคร

⚠️ จดจำ PIN ให้ดี!

PIN ส่วนตัวใช้ login ทุกครั้ง ระบบไม่เก็บ PIN จริง เก็บเฉพาะรหัสที่เข้ารหัสแล้ว ถ้าลืม ต้องใช้ Recovery Code หรือให้ Super Admin reset ให้

ตัวอย่างหน้าจอ

อีเมล
PIN ส่วนตัว (อย่างน้อย 4 ตัว)
รหัสเชิญ
สมัคร

💡 Recovery Code

หลังสมัครสำเร็จ ระบบจะแสดง Recovery Code — ต้องจดเก็บไว้ เพราะแสดงครั้งเดียว ใช้กู้คืน PIN ในกรณีลืม

2

เข้าสู่ระบบ

ใช้อีเมล + PIN ส่วนตัว

🔐 วิธี Login

  1. เปิดเว็บ biomassstove.vercel.app
  2. กรอก อีเมล ที่สมัครไว้
  3. กรอก PIN ส่วนตัว
  4. กด "เข้าสู่ระบบ"

🔑 ลืม PIN?

  1. กดปุ่ม "ลืม PIN?" ที่หน้า login
  2. กรอก อีเมล ที่สมัครไว้
  3. กรอก Recovery Code ที่จดไว้ตอนสมัคร
  4. ตั้ง PIN ใหม่
  5. กด รีเซ็ต

⚠️ Recovery Code ใช้ได้ครั้งเดียว!

ใช้ Recovery Code reset PIN แล้ว code จะถูกลบทันที หลัง login ด้วย PIN ใหม่ ต้องไปสร้าง Recovery Code ใหม่ทันที (ดูหัวข้อ 6) ถ้าไม่สร้าง → ลืม PIN อีกทีต้องให้ Super Admin reset ให้

3

ตั้งค่าเริ่มต้น

กรอกข้อมูลโครงการ อาสา บ้าน เซนเซอร์

📌 ลำดับที่ต้องทำ

ต้องกรอกเรียงลำดับ เพราะข้อมูลเชื่อมกัน:

โครงการ → อาสา → บ้าน → เซนเซอร์ → ช่วงเก็บข้อมูล

A. เพิ่มโครงการ

แท็บ "โครงการ" → กดปุ่ม + เพิ่มโครงการ

📁

ชื่อโครงการ

เช่น "วิจัยมลพิษเตาชีวมวล ปี 2569"

📝

รายละเอียด

ไม่บังคับ — อธิบายเพิ่มเติม

B. เพิ่มอาสาสมัคร

แท็บ "อาสา" → กดปุ่ม + เพิ่มอาสา

👤

ชื่อ-นามสกุล

ชื่อเต็มของอาสา

📱

เบอร์โทร

ไม่บังคับ — สำหรับติดต่อ

C. เพิ่มบ้านกลุ่มตัวอย่าง

แท็บ "บ้าน" → กดปุ่ม + เพิ่มบ้าน

🏠

ชื่อบ้าน

เช่น "บ้านนายสมชาย ตำบลสะเมิง"

📍

GPS พิกัด

ละติจูด, ลองจิจูด จาก Google Maps

🗺️

ที่อยู่ / จังหวัด / อำเภอ

ไม่บังคับ — แต่ช่วยจัดกลุ่มข้อมูล

📁

โครงการ

เลือกโครงการที่สร้างไว้ในขั้น A

👤

อาสาประจำบ้าน

เลือกอาสาที่สร้างไว้ในขั้น B

💡 GPS สำคัญมาก

ระบบใช้ GPS ตรวจจับว่าอาสาอยู่ที่บ้านไหน เมื่อเปิดหน้าเก็บข้อมูล ถ้าไม่กรอก GPS อาสาต้องเลือกบ้านเอง

D. ผูกเซนเซอร์กับบ้าน

แท็บ "อุปกรณ์" → เลือกเซนเซอร์ → ผูกกับบ้าน

เซนเซอร์ที่ลงทะเบียนแล้ว (BS 001–012) จะแสดงในรายการ กดเลือกแล้วผูกกับบ้านที่ต้องการ

📡

เซนเซอร์

เช่น BS 001, BS 002, ...

🏠

บ้าน

เลือกบ้านที่จะติดตั้งเซนเซอร์นี้

⚠️ เซนเซอร์ 1 ตัว = 1 บ้าน

ถ้าต้องการย้ายเซนเซอร์ไปบ้านอื่น ต้องถอดออกก่อนแล้วผูกใหม่

4

ตั้งช่วงเก็บข้อมูล

กำหนดว่าแต่ละบ้านใช้เตาอะไร ช่วงไหน

🔥 ประเภทเตา

🪵

เตาดั้งเดิม

เก็บข้อมูล baseline
ค่ามลพิษแบบเดิม

🌿

เตาชีวมวล

เก็บข้อมูลเตาใหม่
เทียบกับ baseline

📅 วิธีตั้งค่า (ทีละหลายบ้าน)

  1. ไปแท็บ "กำหนดการ"
  2. กดปุ่ม "ตั้งค่าหลายบ้าน"
  3. เลือกบ้านที่ต้องการ (กดเลือกหลายบ้านได้)
  4. เลือก ประเภทเตา (เตาดั้งเดิม / เตาชีวมวล)
  5. กำหนด วันเริ่ม และ วันสิ้นสุด
  6. กด บันทึก

⚠️ ต้องตั้งก่อนเก็บข้อมูล!

ถ้าไม่มีช่วงเก็บข้อมูล ระบบจะไม่รู้ว่าข้อมูลที่ได้เป็นเตาเก่าหรือเตาใหม่ อาจทำให้วิเคราะห์ผิดพลาด

📌 แผนเก็บข้อมูลทั่วไป

เฟส 1: เตาดั้งเดิม 3 เดือน → เก็บค่า baseline มลพิษเดิม

เฟส 2: เตาชีวมวล 3 เดือน → เทียบว่าเตาใหม่ลดมลพิษได้แค่ไหน

5

อนุมัติข้อมูล

ตรวจสอบแล้วอนุมัติ session ที่เก็บเสร็จ

✅ ขั้นตอน

  1. ไปแท็บ "อนุมัติ" (จะมีตัวเลขรอดูด้านขวา)
  2. ดูรายการ session ที่เก็บเสร็จ
  3. ตรวจสอบข้อมูล: บ้าน, วันที่, จำนวนค่าที่เก็บได้
  4. กด อนุมัติ หรือ ไม่อนุมัติ

📊 สถานะ Session

🔄

กำลังเก็บ

เซนเซอร์ยังทำงาน รอเก็บให้ครบ

✅

เก็บครบ

ได้ข้อมูลครบ 24+ จุด พร้อมอนุมัติ

⚠️

ไม่ครบ

ข้อมูลไม่ครบ — อาจเพราะเซนเซอร์หลุด

💡 อนุมัติทีเดียวหลายรายการ

มีปุ่ม "อนุมัติทั้งหมด" สำหรับอนุมัติรวดเดียวหลาย session ไม่ต้องกดทีละอัน

6

เปลี่ยน PIN / Recovery Code

จัดการความปลอดภัยส่วนตัว

🔑 เปลี่ยน PIN

  1. Login แล้ว กดปุ่ม "🔑 PIN" มุมขวาบน
  2. กรอก PIN ใหม่
  3. กรอก ยืนยัน PIN ใหม่
  4. กด เปลี่ยน

🔄 สร้าง Recovery Code ใหม่

  1. Login แล้ว ไปที่เมนู Recovery Code
  2. กด "สร้าง Recovery Code ใหม่"
  3. ระบบแสดง code ใหม่ → จดเก็บไว้ทันที

⚠️ ต้องสร้าง Recovery Code ใหม่ทุกครั้งที่ reset PIN!

เมื่อใช้ Recovery Code reset PIN แล้ว code เดิมจะถูกลบ ถ้าไม่สร้างใหม่ ลืม PIN อีกทีต้องให้ Super Admin reset ให้

7

จัดการแอดมิน

สำหรับ Super Admin เท่านั้น

👑 สิ่งที่ Super Admin ทำได้เพิ่ม

⬆️

เลื่อนขั้นเป็น Super Admin

แท็บ "แอดมิน" → กดปุ่ม "เลื่อนขั้น"

⬇️

ลดขั้นเป็น Admin ปกติ

ต้องมี Super Admin อย่างน้อย 1 คนเสมอ

🗑️

ลบแอดมิน

ลบผู้ใช้ออกจากระบบ

🔑

เปลี่ยนรหัสเชิญ (Invite Code)

เปลี่ยนรหัสสำหรับสมัครแอดมินใหม่

🔄

Reset PIN ให้คนอื่น

กรณีแอดมินลืม PIN และไม่มี Recovery Code

📌 ประวัติกิจกรรม

ทุกการกระทำของแอดมินทุกคนจะถูกบันทึกไว้ในแท็บ "ประวัติ" ดูได้ว่าใครทำอะไร เมื่อไหร่

🗺️ สรุป Flow การทำงาน

1 สมัครแอดมิน (ครั้งเดียว)
↓
2 ตั้งค่า: โครงการ → อาสา → บ้าน → เซนเซอร์ (ครั้งเดียว)
↓
3 ตั้งช่วงเก็บข้อมูล: เตาเก่า / เตาใหม่ (ตามแผน)
↓
4 อาสาเก็บข้อมูลภาคสนาม (ทุกวัน)
↓
5 แอดมินอนุมัติข้อมูล + ดูผลใน Dashboard
