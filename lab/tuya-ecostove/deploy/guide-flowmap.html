สมัคร + Login + Recovery

ขั้นตอนการเข้าสู่ระบบแอดมิน

Flow: สมัครใหม่

Flow: Login

เปิดหน้า Admin → กรอก Email + PIN → ✅ เข้าสู่ระบบ

Flow: ลืม PIN

ตั้ง Collection Period

กำหนดว่าแต่ละบ้านเก็บข้อมูลเตาอะไร ช่วงไหน — สำคัญมาก!

Flow: ตั้งค่าหลายบ้านพร้อมกัน

Tab "กำหนดการ" → "ช่วงเก็บข้อมูลแต่ละบ้าน" → ☑ เลือกบ้านจาก Checkbox → กด "📅 ตั้งค่าที่เลือก" → เลือกประเภทเตา + วันที่ → ✅ บันทึก

🚨 ถ้าไม่ตั้ง Collection Period → ระบบจะไม่รู้ว่าข้อมูลเป็นเตาเก่าหรือเตาใหม่ → Dashboard เปรียบเทียบไม่ได้
