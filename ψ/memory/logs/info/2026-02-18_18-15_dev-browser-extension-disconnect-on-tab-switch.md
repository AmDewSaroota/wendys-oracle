# Dev Browser Extension — หลุดเมื่อสวิทช์จอ/tab

Dev Browser Extension mode หลุด connection ทุกครั้งที่ DewS สวิทช์จอ/tab เพราะ Chrome debugger detach เมื่อ tab ไม่ active

สำหรับงานที่ต้องทำหลาย step ใน Chrome ของ DewS ควรเตรียม prompt + ไฟล์ให้ DewS ทำเองจะเร็วกว่า
