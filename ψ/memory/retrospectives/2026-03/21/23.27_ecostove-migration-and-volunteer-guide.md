## Lessons Learned

1. **Supabase Transfer Project preserves everything** — URL, keys, data ไม่เปลี่ยน แค่ต้อง invite target org member ก่อน
2. **Always verify auth params when recreating endpoints** — อย่าพึ่งความจำ ให้อ่านโค้ดจริง
3. **Field-facing docs need domain expert review** — คู่มืออาสาต้องผ่าน DewS ที่จับเครื่องจริงเสมอ AI ไม่มีทาง physical knowledge
