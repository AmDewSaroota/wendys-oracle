## Lessons Learned

- **Supabase Auth signOut + localStorage clear ต้องระวังลำดับ** — ถ้าลบ session ก่อน onAuthStateChange จะไม่ fire → ต้อง manual switch view
- **ใช้ existing UI patterns** — เช็ค showConfirm/showModal ก่อนใช้ browser native dialogs เสมอ
- **Input width สำหรับ time** — `w-24` แคบเกินสำหรับ time input ที่แสดง AM/PM, ใช้ `w-36` เป็นอย่างน้อย
