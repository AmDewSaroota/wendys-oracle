## Lessons Learned

- **sync_config.active_days blocks weekends silently** — field volunteers have no way to know. Rest day warning on volunteer page is essential UX.
- **MT15 = laser PM, MT29 = infrared PM** — same form factor, different sensor hardware. DewS has MT29, not MT15.
- **Tuya API quota: 1 cron cycle = 1 + N calls** (N = online devices). Not 1 call total. During 130-min session at 6-min interval = ~22 status calls per device.
- **Chinese IoT manufacturers rarely publish detailed datasheets** — community testing and direct purchase are more reliable than spec sheets.
