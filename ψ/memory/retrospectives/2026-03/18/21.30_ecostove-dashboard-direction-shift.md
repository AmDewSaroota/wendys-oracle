## Lessons Learned
- **PostgreSQL NULL trap**: `col NOT LIKE 'pattern'` ไม่จับ NULL → ต้องใช้ `OR col IS NULL` เสมอ
- **Dashboard narrative > data completeness**: อาจารย์ไม่ได้อยากเห็นข้อมูลครบ แต่อยากเห็น "story" ว่าเตาใหม่ดีแค่ไหน
- **รอ stakeholder feedback ก่อน detailed plan**: เวลาทำ dashboard ให้คนอื่นใช้ ควรเอา rough mockup ให้ดูก่อน ค่อย plan รายละเอียด
