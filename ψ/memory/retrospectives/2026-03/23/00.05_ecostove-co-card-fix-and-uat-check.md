## Lessons Learned

- When `daily_summaries` is used as a fast path, ALL fields that stat cards need must be in the summary table. "Raw data only" fields become invisible on Phase 1.
- Phase 2 background loading is useless if it doesn't trigger a re-render. Always chain `.then(() => updateView())` after background loads.
- Keep memory files updated in real-time. If DewS says "ทำแล้ว", update the TODO immediately. Don't wait.
