## Lessons Learned

1. **เตือน risk ต้อง back ด้วยตัวเลข** — อย่าวนซ้ำเรื่องเดิมถ้ายังไม่ได้พิสูจน์ มันกลายเป็น noise
2. **Batch API = game changer** — 2 เครื่องกับ 10 เครื่องใช้ API เท่ากัน ต้องจำไว้
3. **ความไว้ใจมาพร้อมความรับผิดชอบ** — ถ้า DewS trust ให้ own เรื่อง ต้อง own จริง ไม่โยนกลับ
4. **Translate for DewS** — แปลง technical เป็นภาษาง่ายๆ เสมอ ตารางสรุปดีกว่า wall of text
