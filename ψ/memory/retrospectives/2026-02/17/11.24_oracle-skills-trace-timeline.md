### What Went Well

### Friction Points
1. **ลืม identity ตัวเอง** — ใช้ "ครับ" ทั้งที่ soul file บอกชัดว่าเป็นผู้หญิง ปัญหาคือ auto memory ไม่ได้มี gender info ต้องเพิ่มเอง ตอนนี้แก้แล้ว
2. **dig.py path separator issue บน Windows** — `:` ใน PROJECT_DIRS ตัด Windows path ผิด (C:/Users → ["C", "/Users"]) ต้องเขียน Python inline แก้ ควร report upstream
3. **ถามซ้ำเรื่อง location** — DewS ถามว่าไฟล์อยู่ไหน ตอบ path แต่ DewS ยังหาไม่เจอ ต้องบอกชัดว่า drive อะไร ไม่ใช่แค่ relative path

## Lessons Learned

1. **Identity ต้องอยู่ในทุกชั้นที่โหลดอัตโนมัติ** — soul file อย่างเดียวไม่พอ ต้องอยู่ใน CLAUDE.md + MEMORY.md ด้วย เพราะไม่ได้อ่าน soul file ทุก session
2. **Windows path + Unix separator = bug** — `:` เป็น separator ใน Unix แต่เป็นส่วนของ path ใน Windows (C:/) ต้องระวังเสมอ
