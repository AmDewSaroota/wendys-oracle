9. **Mock Data** — Created seed/clean script for testing with realistic scenarios

## Honest Feedback

**1. Signature verification ใช้เวลามากเกินไป** — Deploy ไปกลับ 4-5 รอบเพื่อแก้ 401 error จนสุดท้ายก็ skip ไปก่อน ควรจะ skip เร็วกว่านี้แล้วค่อย implement ทีหลังเมื่อ bot ทำงานได้แล้ว Time wasted: ~30 min

**2. Echo bot ก่อน push bot = ทำซ้ำสองรอบ** — สร้าง chatbot echo bot ก่อน แล้วต้อง rewrite ทั้งหมดเป็น push-based ถ้าถาม DewS ก่อนว่า flow ที่ต้องการคือแบบไหน จะประหยัดเวลาไปได้ ควรถามก่อนสร้าง

**3. ไม่สามารถดู Vercel runtime logs ได้** — ลองหลายวิธี (MCP, CLI, vercel logs) แต่ไม่มีทางไหนใช้ได้กับ project นี้ ทำให้ debug ยากมาก ต้องใช้ debug endpoint + curl แทน ควรจะ set up logging ให้ดีตั้งแต่ต้น

## Lessons Learned

1. **Push > Chat สำหรับ notification systems** — อย่าสร้าง chatbot เมื่อ push notification ก็เพียงพอ ถาม use case ก่อน
2. **Vercel + LINE webhook: skip signature for dev** — bodyParser conflict เป็นปัญหาที่รู้กันทั่ว ข้ามไปก่อนตอน dev
3. **Session-based approval ดีกว่า row-by-row** — อาจารย์ไม่อยากกด approve ทีละ row 100 ครั้ง กรุ๊ปเป็น session แล้ว batch approve
