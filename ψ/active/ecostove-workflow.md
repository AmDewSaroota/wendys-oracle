# EcoStove - Full Workflow Design

## โครงสร้าง

```
1 อาสาสมัคร
    └── 5 บ้านตัวอย่าง
            └── 1 Device (ZN-MT29) ต่อบ้าน
```

## การเก็บข้อมูลต่อวัน

### 1 Session ประกอบด้วย:
1. **ก่อนใช้เตา** (Baseline): 10 นาที
2. **ระหว่างใช้เตา** (During): 2 ชั่วโมง

### ต่อวัน:
- 2 sessions ต่อบ้าน
- รวม: 5 บ้าน × 2 sessions = 10 sessions/วัน/อาสาสมัคร

## อุปกรณ์

| อุปกรณ์ | หน้าที่ |
|---------|---------|
| ZN-MT29 | วัดค่าอากาศ |
| มือถืออาสาสมัคร | เก็บข้อมูล + อัปโหลด |

## Server

- **ตอนนี้**: Supabase (จำลอง)
- **ภายหลัง**: Server จริงของลูกค้า
